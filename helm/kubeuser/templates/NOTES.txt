3. Create your first user with x509 authentication:
   cat <<EOF | kubectl apply -f -
   apiVersion: auth.openkube.io/v1alpha1
   kind: User
   metadata:
     name: example-user
   spec:
     auth:
       type: x509              # Authentication type (x509 or oidc)
       ttl: "720h"             # Certificate lifetime (30 days)
       autoRenew: true         # Enable automatic renewal
       renewBefore: "240h"     # Renew 10 days before expiry
     clusterRoles:
       - existingClusterRole: view
   EOF

4. Create a user with minimal configuration (defaults to x509):
   cat <<EOF | kubectl apply -f -
   apiVersion: auth.openkube.io/v1alpha1
   kind: User
   metadata:
     name: simple-user
   spec:
     clusterRoles:
       - existingClusterRole: edit
   EOF

5. Check user status:
   kubectl get users
   kubectl describe user example-user

6. Retrieve the generated kubeconfig:
   kubectl get secret example-user-kubeconfig -o jsonpath='{.data.kubeconfig}' | base64 -d > example-user.kubeconfig
